# 1.0 引言

- php中的转义符号为`\`
- php中的变量在单引号内部无法转变为变量
- heredoc内容简记

看例子：

```php
$params="请注意看下面的内容，即使变量被保存在单引号内部，依然可以执行";
print <<< START

print '$params';

START;
```

这里的`START`可以是任何你喜欢的单词，**建议全部大写**
